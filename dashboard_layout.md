# Power BI Dashboard Layout

## Page Title
Sales Performance Dashboard

## Visual 1: KPI Cards
- Total Revenue
- Total Orders
- Total Units
- Average Order Value

## Visual 2: Monthly Revenue Trend
- Line chart
- Axis: month_name
- Values: total_revenue

## Visual 3: Revenue by Product Category
- Clustered bar chart
- Axis: category
- Values: total_revenue

## Visual 4: Regional Performance Map
- Filled map or heat map
- Location: region_name
- Color saturation: total_revenue

## Visual 5: Top Products Table
- Columns: product_name, category, total_revenue, total_units, total_orders

## Visual 6: Quarterly Performance Comparison
- Column chart
- Axis: quarter_label
- Values: revenue

## Recommended Slicers
- quarter_label
- category
- region_name
- customer_segment
- sales_channel
